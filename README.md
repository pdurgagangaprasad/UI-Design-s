# Myntra UI Design Clone (Figma)

## Overview
This project is a UI design clone inspired by Myntra, created using Figma. It focuses on recreating a modern e-commerce shopping experience with clean layouts, attractive visuals, and user-friendly interface design.

## Tool Used
- Figma

## Features
- E-commerce home page design
- Modern navigation bar
- Product showcase layouts
- Category sections
- Product cards
- Clean UI components
- User-friendly shopping experience design

## Purpose
The purpose of this project is to practice UI/UX design skills, understand e-commerce design patterns, and improve Figma design capabilities.

## Disclaimer
This is a learning and design practice project. It is not affiliated with or an official product of Myntra.
